Hi! I'm Yunhai, a senior student at Nanjing University, majoring in Computer Science & Technology. I'm also a member of the National Elite Program of Computer Science.

I'm broadly interested in Robotics, Reinforcement Learning, and Computer Vision. I aim to build intelligent systems that are robust and generalizable.

Currently I'm working with Prof. Alberto Sangiovanni-Vincentelli and Dr. Xiangyu Yue on Domain Generalization. Previously, I was also fortunate to work in Stanford Artificial Intelligence Lab (SAIL), supervised by Prof. Kenneth Salisbury, and in Leaning And Mining from DatA (LAMDA) Group, Nanjing University, supervised by Prof. Yang Yu.

Projects

Design and Control of Roller Grasper V3 for In-Hand Manipulation

[to be submitted to Science Robotics] [related projects: V1, V2]

A hierarchical manipulation pipeline for transforming an arbitrary object from an arbitrary initial pose to an arbitrary target pose, which consists of a high-level planner that searches for a feasible path with waypoints, and a low-level control policy that navigates the object in between the adjacent waypoints.

Learning Quadruped Robot Control with Deep Reinforcement Learning

[coming soon]

A moddel-free method for quadruped robot control, which solves partially observable Markov decision process (POMDP) by deep reinforcement learning.

Multi-Cloud Storage

[demo] [presentation] [slides]

Client-server programs for multi-cloud file storage (i.e., distributed storage on multiple servers), which achieves reliable distributed storage with file recovery mechanism based on Jerasure erasure coding, and efficient file access based on parallel file transfer.

MyPaint

[demo] [documentation]

An interactive graphics system based on 2D graphics algorithms, which supports the generation and transformation of primitives, and basic edit and file operations.
